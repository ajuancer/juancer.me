404 - page not found

The 404 HTTP status code is a standard response code that informs the requested page was not found. This is a static web page which means that the error is most probably from one of the following errors:

  • the url is wrongly typed (i usually use readable urls so check for misspelling),
  • i have changed the name of the file,
  • deleted or changed its visibility
